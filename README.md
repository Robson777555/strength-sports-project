Projeto Strongman e Arm Wrestling

Este projeto é um website estático dedicado ao mundo do Strongman e Arm Wrestling, apresentando informações sobre os maiores atletas dessas modalidades,
uma seção de contato e uma funcionalidade de pesquisa de atletas por peso.

Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias web:

•
HTML5: Para a estruturação do conteúdo das páginas.

•
CSS3: Para a estilização e responsividade do layout.

•
JavaScript: Para a interatividade na página de atletas (funcionalidade de pesquisa por peso).

Funcionalidades

O website possui as seguintes funcionalidades:

•
Página Inicial (index.html): Apresenta uma introdução ao mundo do Strongman e Arm Wrestling,
destacando alguns dos maiores atletas com breves descrições e imagens.

•
Página de Atletas (atletas.html): Lista detalhadamente os atletas, incluindo foto,
categoria (Strongman ou Arm Wrestling), peso e uma breve descrição. Possui uma funcionalidade de pesquisa que permite filtrar atletas por peso.

•
Página de Contato (contato.html): Fornece informações de contato como e-mail, telefone e localização, além de uma breve descrição sobre o especialista.

•
Design Responsivo: O layout do site é adaptável a diferentes tamanhos de tela, proporcionando uma boa experiência de usuário em dispositivos móveis e desktops.
